TYPE 사내 업무
TECH STACK HTML, CSS (Grid), Vanilla JS, GitHub Pages
TROUBLESHOOTING & IMPACT

최근 AI 코딩 비서에 의존하여 레이아웃을 생성하는 비전문가 사용자들이 많아졌으나, 복잡한 CSS Grid 규칙(Grid-template, Area 병합 등)을 세부 조정하거나 AI가 생성한 잘못된 코드를 디버깅하는 일은 비전문가에게 매우 높은 장벽이었습니다. 이러한 문제를 해결하기 위해, 직관적인 시각적 조작(셀 추가, 병합, 간격 조정 등)만으로 정교한 그리드 구조를 제작할 수 있는 도구를 개발했습니다. 에디터 내부에서 엄격한 CSS 규격을 상시 검증하므로, 지식이 없어도 규칙이 깨지지 않는 프로덕션 레벨의 깨끗한 CSS/HTML 코드를 즉시 내보낼(Export) 수 있습니다. 이를 통해 비전문가도 AI 코드에만 의존하지 않고 안전하고 완전한 레이아웃 결과물을 직접 통제하고 획득할 수 있는 개발 편의성을 제공합니다.
